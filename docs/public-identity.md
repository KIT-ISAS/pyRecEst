# Public Identity

Use the same public identity across documentation, package metadata, releases,
and citations:

| Surface             | Preferred form                                       |
|---------------------|------------------------------------------------------|
| Project name        | `PyRecEst`                                           |
| Import/package name | `pyrecest`                                           |
| Repository          | `FlorianPfaff/PyRecEst`                              |
| Citation title      | `PyRecEst: Recursive Bayesian Estimation for Python` |

Avoid mixed variants such as `pyRecEst` in new public material unless referring
to historical artifacts. Consistent naming helps package discovery, citation
hygiene, and user support.
